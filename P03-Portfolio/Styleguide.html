Style Guide

Interactive Elements Text Elements Combined Elements

Navigation Button

CSS selector:

  • .button-nav

Sample code:

<a class="button-nav" href="#">nav button</a>

Rendered element:

nav button

In-text link

CSS selector:

  • .in-text-link

Sample code:

<a class="in-text-link" href="#">here's the link</a>

Rendered element:

Within a paragraph there will come a point at which there is a link. At that point, it'll say here's the link and when you click on it, it'll do something! Who knows what? The possibilities are endless!

Read More Button

CSS selector:

  • .button-read-more

Sample code:

<a class="in-text-link" href="#">here's the link</a>

Rendered element:

tell me more!
